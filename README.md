# Tetris Game

This project is a Java implementation of the classic Tetris game. It includes various components such as game panels, key handlers, and mino types. 

## Directory Structure
- `src/main/java/main` - Contains the main game functionalities.
- `src/main/java/mino` - Contains different mino (tetromino) types.

## How to Build
You can build this project using Maven. Make sure you have [Maven](https://maven.apache.org/) installed.

## Running the Game
After building, run the `main` class to start the game.

Enjoy playing Tetris!
